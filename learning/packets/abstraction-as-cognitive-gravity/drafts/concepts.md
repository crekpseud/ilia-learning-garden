# Candidate Concepts

## Abstraction As Cognitive Gravity

- Status: new candidate
- Definition: A metaphor where abstractions act as cognitive attractors that gather many concrete implementations, examples, and practices into stable conceptual basins.
- Why it matters: It reframes abstraction as a structuring force of thought rather than only a technique for hiding detail.
- Possible aliases: abstraction as gravity, cognitive gravity, abstraction as attractor, cognitive potential well
- Related existing notes: [[wiki/concepts/object-thinking]], [[wiki/synthesis/object-thinking-and-the-digital-garden]], [[wiki/concepts/first-principles-reasoning]]
- Promotion recommendation: yes, after user review
- If existing, suggested patch: N/A

## Entropy-Abstraction Cycle

- Status: new candidate
- Definition: A proposed pattern in software history where new capabilities increase architectural and cognitive entropy, after which new abstractions emerge to make the expanded state space tractable.
- Why it matters: It provides the dynamic background for the gravity metaphor: abstraction appears after entropy expansion creates pressure for cognitive compression.
- Possible aliases: entropy-compression cycle, abstraction-compression cycle, software entropy cycle
- Related existing notes: [[wiki/synthesis/object-thinking-and-the-digital-garden]], [[wiki/concepts/object-thinking]], [[wiki/synthesis/original-thought-as-active-reconstruction]]
- Promotion recommendation: yes, maybe as part of the synthesis before a standalone concept
- If existing, suggested patch: N/A

## Cognitive Entropy

- Status: new candidate
- Definition: The experienced uncertainty and conceptual load created when a person must reason across many possible concepts, interactions, technologies, or system states.
- Why it matters: It names the pressure that abstractions, learning gardens, and agentic skills try to make tractable.
- Possible aliases: human cognitive entropy, conceptual entropy, cognitive state-space entropy
- Related existing notes: [[wiki/concepts/desirable-difficulties]], [[wiki/concepts/the-ritualization-trap]], [[wiki/synthesis/original-thought-as-active-reconstruction]]
- Promotion recommendation: maybe
- If existing, suggested patch: N/A

## Architectural Entropy

- Status: new candidate
- Definition: The growth of possible configurations, interactions, and operational states inside a technological ecosystem.
- Why it matters: It distinguishes system-level state-space expansion from the human experience of cognitive entropy.
- Possible aliases: state-space entropy, system entropy, ecosystem entropy
- Related existing notes: [[wiki/concepts/object-thinking]], [[wiki/synthesis/object-thinking-and-the-digital-garden]]
- Promotion recommendation: maybe
- If existing, suggested patch: N/A

## Skill As First-Class Software Artifact

- Status: new candidate
- Definition: The idea that agent skills may become durable, reusable units of expertise comparable to classes, services, containers, or components.
- Why it matters: It connects the garden's agent skill work to a broader theory of software evolution and agentic AI.
- Possible aliases: skill artifact, reusable expertise unit, agentic skill artifact
- Related existing notes: [[wiki/concepts/knowledge-gardener]], [[wiki/concepts/note-titles-as-apis]]
- Promotion recommendation: maybe, likely after the agentic AI packet is touched
- If existing, suggested patch: N/A

## Map Of Curiosity

- Status: new candidate
- Definition: An optional untrusted artifact that maps recurring questions, curiosity trails, nearby trusted notes, and possible next packets before deciding what deserves promotion.
- Why it matters: It prevents long conversations from being reduced only to their final answer and supports learning paths that grow through repeated questioning.
- Possible aliases: curiosity map, learning trail map, inquiry map
- Related existing notes: [[wiki/concepts/knowledge-gardener]], [[wiki/concepts/playful-knowledge-practice]], [[wiki/concepts/digital-garden]]
- Promotion recommendation: maybe, after a few more real uses
