# Keep learning packets inside the vault

Learning packets will live inside the Obsidian vault, but outside the trusted wiki. This keeps the learning process local-first, inspectable, linkable, and recoverable while preserving a clear boundary between in-progress learning artifacts and promoted trusted knowledge.

**Considered Options**

- Store learning packets inside the Obsidian vault.
- Store learning packets in the project workspace and export only final notes.
- Store learning packets in app or database storage.

**Consequences**

The vault will contain both trusted and untrusted material, so directory boundaries and metadata must make trust state obvious.
