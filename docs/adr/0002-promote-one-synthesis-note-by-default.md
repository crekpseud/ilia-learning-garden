# Promote one synthesis note by default

Each approved learning packet will promote exactly one synthesis note by default, plus zero or more concept notes only when those concepts are reusable beyond the packet. This keeps the trusted wiki from inflating into many low-value generated notes while still allowing important concepts to compound over time.

**Considered Options**

- Promote one synthesis note only.
- Split each packet into many atomic concept notes.
- Promote one synthesis note plus selected reusable concept notes.

**Consequences**

The first version needs a review step for deciding which candidate concepts deserve promotion, but the default vault growth remains conservative.
