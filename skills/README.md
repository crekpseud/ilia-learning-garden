# Agent Skills

Reusable agent skills for operating this learning garden.

These skills are Markdown instruction bundles. They are intentionally tool-light so they can be read by Codex, GitHub Copilot, or another agent.

## Available Skills

- [[skills/learning-garden-capture/SKILL|Learning Garden Capture]] - Capture reusable insights from an AI-assisted work or learning session into the garden inbox without promoting them into the trusted wiki.
